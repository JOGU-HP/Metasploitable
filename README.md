# Metasploitable

En este repositorio, se documentan las practicas de explotacion a una maquina vulnerable (Metasploitable)

Objetivos:
-Practicar hablidades con Kali y la explotacion de vulnerabilidades
-Aprender a levantar un entorno vulnerable, configurarlo y explotarlo
-Utilizar herramientas de Kali para explotar las vulnerabilidades encontradas
-Documentar como se explotan las vulnerabilidades y los pasos para cada una de ellas
-Realizar documentaciones claras, entendibles que ayuden a mi formacion hacia el pentesting

Inicio: 18 de enero del 2026
